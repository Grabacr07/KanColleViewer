KanColleViewer PluginAnalyzer
--

提督業も忙しい！ (KanColleViewer) のプラグイン開発を支援する Visual Studio 2015 用アナライザーです。  
プラグインのコードがプラグイン開発のルールに則っているかどうかを分析し、問題があれば修正方法を提示します。
